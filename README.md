# Bamazon
## A Shopping Simulation Created with Node.JS and MySQL Database 

**NPM Packages required:**
* Inquirer 
* mysql
* console.table


**To Use:**

1. Run: *node bamazonCustomer.js* 
2. Follow prompts to complete purchases
3. Use **Q** to exit program at any time
